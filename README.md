# Ostyn's Dashboard

Personal wellness tracker — habits, workouts, protein, water, and body measurements.

## Deploy to Vercel (5 min)

1. Go to https://github.com and create a free account if you don't have one
2. Create a new repository called `ostyn-dashboard`
3. Upload all these files (drag and drop works)
4. Go to https://vercel.com and sign in with GitHub
5. Click "Add New Project" → select your repo → click Deploy
6. Done! You'll get a live URL like `ostyn-dashboard.vercel.app`

## Add to iPhone Home Screen

1. Open your dashboard URL in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add" — it'll appear as an app icon

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
